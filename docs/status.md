# Project Status

Digital Lineage Objects (DLOS) are currently an experimental
architecture and reference implementation.

The goal of this repository is to explore a model in which
digital artifacts can preserve their identity, authorship,
and lineage independent of platforms or storage systems.

The current stage focuses on:

- defining a minimal object container
- demonstrating artifact self-verification
- establishing a lineage model
- providing a reference implementation

The specification should not yet be considered stable.

Future development will depend on experimentation,
community feedback, and potential interest from
archival, cultural, and research institutions.

The long-term direction of the project remains open.
