# Project Status

DLOS is an experimental protocol prototype.

This repository contains:

- a draft protocol specification
- a minimal reference implementation
- example artifacts demonstrating the object structure

The implementation is intended to illustrate the core mechanics of
object identity, integrity verification, and lineage referencing.

It should not be considered production-ready.

Future work may include:

- formal specification refinement
- independent implementations
- additional verification tooling
- ecosystem services
